# Customer Segmentation and Churn Prediction

## 📌 Project Overview

This project focuses on analyzing customer behavior and predicting customer churn using Machine Learning techniques. The primary objective is to help businesses identify valuable customer segments and predict customers who are likely to discontinue services, enabling better retention strategies.

---

## 🎯 Objectives

- Perform customer segmentation based on customer characteristics.
- Analyze customer behavior patterns.
- Predict customer churn using Machine Learning models.
- Generate actionable business insights for customer retention.
- Visualize key trends and patterns through data analysis.

---

## 📊 Dataset

**Dataset:** Telco Customer Churn Dataset

The dataset contains customer information such as:

- Customer demographics
- Contract details
- Internet services
- Payment methods
- Monthly charges
- Total charges
- Churn status

## 📈 Project Workflow

### 1. Data Collection
- Load customer churn dataset.

### 2. Data Preprocessing
- Handle missing values.
- Remove duplicates.
- Encode categorical variables.

### 3. Exploratory Data Analysis (EDA)
- Customer distribution analysis.
- Churn rate analysis.
- Correlation analysis.
- Data visualization.

### 4. Customer Segmentation
- Apply K-Means Clustering.
- Identify customer groups based on behavior.

### 5. Churn Prediction
- Train Machine Learning models.
- Predict customers likely to churn.

### 6. Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

### 7. Business Insights
- Customer retention recommendations.
- Identification of high-risk customer segments.

---

## 🤖 Machine Learning Models

### Customer Segmentation
- K-Means Clustering

### Churn Prediction
- Logistic Regression
- Random Forest Classifier
